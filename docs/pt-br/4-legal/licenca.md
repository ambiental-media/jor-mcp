## Termos e Condições

A Licença Apache 2.0 é uma licença de software livre permissiva escrita pela Apache Software Foundation. Ela permite que você:
- **Uso Comercial:** Use o software comercialmente.
- **Modificação:** Modifique o código-fonte.
- **Distribuição:** Distribua o software e quaisquer modificações.
- **Sublicenciamento:** Conceda sublicenças a terceiros.
- **Concessão de Patentes:** Concede explicitamente direitos de patente aos colaboradores.

### Requisitos Chave:
- **Aviso de Direitos Autorais:** Você deve reter todos os avisos de direitos autorais, patentes, marcas registradas e atribuições da fonte original.
- **Indicação de Alterações:** Quaisquer arquivos modificados por você devem conter avisos claros indicando que você realizou as alterações.
